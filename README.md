# 重返未来1999 - 抽卡统计

一个用于记录和统计《重返未来1999》抽卡数据的项目，包含抽卡记录和角色信息，提供网页可视化展示。

## 项目结构

```
Reverse-1999/
├── characters/          # 角色数据目录
│   ├── 2/gainIds.josn   # 2星角色列表
│   ├── 3/gainIds.josn   # 3星角色列表
│   ├── 4/gainIds.josn   # 4星角色列表
│   ├── 5/gainIds.josn   # 5星角色列表
│   └── 6/gainIds.josn   # 6星角色列表
├── poolType/            # 卡池类型目录
├── pageData.josn        # 抽卡记录数据
├── index.html           # 抽卡统计网页
└── README.md            # 项目说明
```

## 数据文件说明

### characters/ 目录

存放各稀有度角色的ID和名称映射，格式为：

```
角色ID:角色名称,
```

例如：

```
3146:小瑞安侬,
3154:银装小春雀儿,
```

### poolType/ 目录

存放各卡池类型的名称映射，目录名对应 `poolType` 字段值，文件名即为卡池名称。

### pageData.josn

存放抽卡记录，每条记录包含：

| 字段 | 类型 | 说明 |
|------|------|------|
| summonType | string | 抽卡类型，`"1"` 为单抽，`"10"` 为十连 |
| poolName | string | 卡池名称 |
| poolType | number | 卡池类型ID |
| poolId | number | 卡池ID |
| gainIds | array | 本次抽卡获得的角色ID列表 |
| createTime | string | 抽卡时间，格式 `YYYY-MM-DD HH:MM:SS` |
| luckyBagIds | array | 幸运袋获得的角色ID列表（通常为空） |

## 名词解释

### 稀有度等级

游戏中角色稀有度分为5个等级：

- **6星**：最高稀有度角色，强度通常最高
- **5星**：高级稀有度角色
- **4星**：中级稀有度角色
- **3星**：普通稀有度角色
- **2星**：最低稀有度角色

### 卡池

游戏中的抽卡池，不同卡池有不同的UP角色和概率分布。

### 十连

一次性抽取10次，通常有保底机制（第10次必定获得3星及以上角色）。

### 歪卡

抽到非UP角色的6星角色。例如，在"故事的新啼"卡池中抽到小瑞安侬，但该卡池的UP角色并不是小瑞安侬，即为歪卡。

## 抽卡统计网页

打开 `index.html` 即可查看抽卡统计，网页会自动读取数据文件并展示：

- **统计数据**：六星获取数、歪卡次数、累计抽数
- **卡池列表**：每个卡池的抽数和获得的六星角色
- **抽数计算**：按时间顺序精确计算获得六星角色所花费的抽数

## 运行方式

1. 使用浏览器直接打开 `index.html`（需要本地服务器支持）
2. 或启动本地服务器：

```bash
python -m http.server 8001
```

然后访问 `http://localhost:8001/index.html`

## 更新数据

只需更新 `pageData.josn` 文件中的抽卡记录，刷新网页即可看到最新统计结果。
